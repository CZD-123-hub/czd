## 三、知网导出操作建议

1. 打开知网高级检索。
2. 将“推荐检索词”复制到主题或篇名字段。
3. 优先选择“篇名”检索，结果不准确时再改为“主题”。
4. 找到目标文献后，勾选该文献。
5. 点击“导出/参考文献”。
6. 格式选择 GB/T 7714。
7. 将导出的参考文献粘贴到论文参考文献章节。

## 四、核对重点

1. 作者顺序是否一致。
2. 题名是否完全一致。
3. 期刊名是否完整。
4. 年份、卷号、期号、页码是否完整。
5. DOI 是否存在并且没有多余空格。
6. 英文文献作者大小写是否统一。
7. 参考文献编号是否与正文引用编号一致。

## 五、检索不到时的处理方式

1. 先删掉副标题或标点，只搜主标题。
2. 加第一作者姓名缩小范围。
3. 改用万方、维普、百度学术或期刊官网检索。
4. 若仍检索不到，建议替换为同主题、可检索、信息完整的文献。
